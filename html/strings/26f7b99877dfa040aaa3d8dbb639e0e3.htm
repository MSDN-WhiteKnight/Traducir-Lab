Original string:
Reputation recalc can no longer be triggered. If your rep is off after a post deletion, allow 5 minutes for it to sync up. If your reputation is off for another reason, let us know on meta.stackexchange.com. We have disabled this ability because reputation should be in sync without any user intervention. If it's not in sync for more than 5 minutes, it's a bug. We need to investigate that bug and a recalc hinders fixing the actual cause.
